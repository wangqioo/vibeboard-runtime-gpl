# codex_chibi

Generated Codex Buddy character pack based on the user-provided reference image.

All GIFs are `120x120` on a pure black background and are intended to be shown at native size on the 320x240 device screen.

State mapping:

- `sleep.gif`: sleeping
- `idle_0.gif`: idle blink
- `busy.gif`: typing/coding
- `attention.gif`: work supervisor / attention
- `celebrate.gif`: completed
- `dizzy.gif`: error shrug
- `heart.gif`: fixing bug / healthy

The folder name remains `bufo` so existing `main.lua` configuration can keep using the default character path.
